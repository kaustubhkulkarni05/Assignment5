# Team 7 Question Answer Node.js Application


## Files

Team 7 Question Answer Node.js application has files as below:


*   app.js

    This file contains the server side JavaScript code for your application written using the Node.js API

*   views/

    This directory contains the views of the application. It is required by the express framework and jade template engine in this qa application.It consists of two files index.jade and layout.jade.

*   public/

    This directory contains public resources of the application. It is required by the express framework in this qa application.It comprises of sources to css files,images and js files.

*   package.json

    This file is required by the Node.js environment. It specifies this Node.js project name, dependencies, and other configurations of qa application.
